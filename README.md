# 喜羊羊与灰太狼小游戏 README

## 游戏介绍
这是一款基于Phaser.js开发的"喜羊羊与灰太狼"主题横版闯关小游戏。玩家可以选择不同的角色（喜羊羊、美羊羊、灰太狼）进行游戏，通过跳跃和移动躲避障碍物、击败敌人，最终到达终点。

## 游戏功能
- 角色选择系统：可在开始界面选择喜羊羊、美羊羊或灰太狼
- 丰富的游戏场景：包含多个平台、金币和障碍物
- 敌人系统：包含多种不同行为模式的敌人
- 技能系统：收集金币后可释放"半月斩"技能
- 进度跟踪：实时显示当前闯关进度
- 音效系统：包含背景音乐和各种交互音效

## 操作说明
- ↑ 键：跳跃
- ← → 键：左右移动
- 空格键：开始游戏 / 释放"半月斩"技能（获得金币后）

## 游戏目标
控制所选角色，躲避障碍物和敌人的攻击，到达终点金币获取特殊技能，最终到达终点通关。

## 开发环境
- 基于Phaser.js游戏框架开发
- 可在支持HTML5的浏览器中运行

## 文件结构
- `chuangguan/index.html`：游戏主页面，包含游戏逻辑和界面布局
- `res/`：资源文件夹，包含游戏所需的图片和音效文件
- `project.config.json` 和 `project.private.config.json`：项目配置文件

## 游戏特色
- 角色选择系统，提供不同角色外观
- 多样化的敌人类型，包括固定路径移动的敌人和发射炮弹的敌人
- 技能系统增加游戏趣味性和策略性
- 精美的视觉效果和进度提示
- 响应式设计，适配不同屏幕尺寸

## 如何开始
1. 打开`chuangguan/index.html`文件
2. 按空格键进入角色选择界面
3. 选择喜欢的角色并点击"确认选择"
4. 使用方向键和空格键开始游戏

祝玩得愉快！
